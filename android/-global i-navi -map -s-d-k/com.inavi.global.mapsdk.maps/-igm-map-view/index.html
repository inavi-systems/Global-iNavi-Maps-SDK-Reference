IgmMapView

class IgmMapView @JvmOverloads constructor(context: Context, attrs: AttributeSet? = null, defStyleAttr: Int = 0, options: IgmMapOptions? = null) : FrameLayout

View class that renders the map. Add this view to your screen and call IgmMapView.getMapAsync to receive an IgmMap instance.

When using this view directly, call the corresponding methods according to the lifecycle of the hosting Activity or Fragment:

  • IgmMapView.onCreate

  • IgmMapView.onStart

  • IgmMapView.onResume

  • IgmMapView.onPause

  • IgmMapView.onStop

  • IgmMapView.onDestroy

  • IgmMapView.onSaveInstanceState

  • IgmMapView.onLowMemory

    If you use IgmMapFragment, these lifecycle integrations are handled automatically.
